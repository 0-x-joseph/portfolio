## FreelanceOS

Client portal for freelancers built during the **Bolt $1M Hackathon**.

**Stack**

- Supabase
- TailwindCSS

## 42SDS

Adaptation of the **Redis SDS (Simple Dynamic Strings)** library for use in the **42 Network curriculum**.

The original SDS implementation used in Redis was refactored to comply with **42 coding standards (Norminette)** so students can use a battle-tested dynamic string library in their projects.

**Focus**

- systems programming
- memory management
- low-level C
