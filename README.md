# suckless-pandoc-site

Minimal personal website inspired by **suckless.org**.

- Static HTML
- Pandoc + Makefile
- No JavaScript
- Black & white
- Light/Dark via prefers-color-scheme

## Requirements

- pandoc
- make

## Build

```bash
make

```

Output is generated in `public/`.

## Deploy

### GitHub Pages

* Set Pages source to `public/`
* Add a `CNAME` file if using a custom domain

## Philosophy

* Write content, not configuration
* Prefer boring tools
* No JavaScript
* HTML should be readable without CSS

This repo is meant to last.
